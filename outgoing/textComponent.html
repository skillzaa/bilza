Text Component

Text is one of the most important comp (component) of bilza library. It is not only different from other components of this library but the "Text" comp in bilza.js is also implemented differently from other libraries.

To understand Text comp we need to understand the following 3 props of the Text comp

  1. "fontSize" prop:

    2. The fontSize prop of the Text comp is the main factor which decided the width and height of Text comp. If you do not need to worry about the width and height of the text content just use fontSize. The resultant width and height of the comp will be what ever it is at that fontSize.

    fontSize Animate Example

  2. "fitToWidth" prop:

    3. Text comp has a boolean (AniBoolean) prop "fitToWidth". This prop by default is set to false.

    When "fitToWidth" is set to false the width of Text comp is treated as "Maximum Width". It means that we can set any value for fontSize prop and it is fine but if the resultant width is larger than comp.width then it is reduced to the given width.

    By default the width (and also height) value of a Text comp are set at 300% thus they have no effect most of the times. To see some effect these values need to be reduced by the user.

  3. "fitToHeight" prop:

    4. Text comp has a boolean (AniBoolean) prop "fitToHeight". This prop by default is set to false.

    When "fitToHeight" is set to false the height of Text comp is treated as "Maximum Height". It means that we can set any value for fontSize prop and it is fine but if the resultant height is larger than comp.height then it is reduced to the given height.

    By default the height (and also width) value of a Text comp are set at 300% thus they have no effect most of the times. To see some effect these values need to be reduced by the user.
