What is Bilza.js

Features of Bilza.js

  1. It is a library / framework to draw on to HTML5 canvas.
  2. Bilza does not have any dependencies, that means you do not need to download anything else other than this library.
  3. It can be refered to as framework since it has a run time.
  4. The output is simple javascript code that can run in any browser.
  5. The library has a core (called the engine) and components. At the moment we have 10-15 components, many more will be added soon.
