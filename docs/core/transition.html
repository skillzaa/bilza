Transition

Components can change their state at any given micro second using transitions. All we have to do is add transition this will give us an object containing all the variables of that component; but with the values set to null. Now we can change the values of which ever variable we want and leave the remaining as null. Once the apply method is called with the appropriate mili second value, the changes in the transition object are merged with the variables of the component.

  1. add(msStart :number):T

  2. insert(f:T):T

  3. apply(msDelta :number)
