Objects contained in Bilza

  1. background :Background;
  2. insert

Public Methods

  1. draw(msDelta :number=0)

    2. This is the main draw function.It takes a millisecond (as argument) and draw on canvas that specific frame.
    When bil.start() is used the same method (bil.draw()) is called with different msDelta in a loop.

  2. len(inMilliSeconds :boolean = true):number

    3. Returns the length of the animation. It can return in seconds or milliseconds

  3. drawEvent(msDelta :number)

    4. This is a method that is run after every draw. The user can override this methos and insert his own code to be run on every draw. 27-june-2022: Not fully implemented yet

  4. dynamicCanvas(widthInPercent:number = 95,heightInPercent :number | null=null):boolean

    5. We use dynamicCanvas method to set the width and height of current canvas according to some percentage of the screen.
    The width and height can not be larger than 100%.
    It is advised to just provide widthInPercent and leave the heightInPercent blank which has null as default value. Once null value for heightInPercent is revieved the height is set dynamically to 16:9 aspect ration.

  5. resizeCanvas(width :number, height :number | null)

    6. We use resizeCanvas method to set the width and height of current canvas according to fixed number of pixels.
    It is advised to just provide width and leave the height blank which has null as default value. Once null value for height is recieved the height is set dynamically to 16:9 aspect ration. At the moment there is no max size (of width or height) for this method.

  6. start():boolean

    7. The bil.start(); will start running the animation.

  7. stop():boolean

    8. The bil.stop(); will stop running the animation.

  8. getLastMsDelta():number

    9. This will return the last millisecond run by draw method.

  9. isRunning():boolean

    10. Returns boolean to show if the current state is running or not.
