AniNumber

AniNumber (Animated Number) is a class that wraps a number and animate it.

Most of the numeric properties of a component is bilza library has AniNumber class. It provides methods like animate, vibrate, jumpBetween etc.

---

Following methods are available

  1. goto
  2. animate
  3. vibrate
  4. jumpBetween
  5. random
  6. oscillate
