Component

A component is what the Bilza Engine consumes. There is no animation without components.
Every component has following properties and objects:

Public Properties

  1. readonly id :string;

    2. Every component has an automatically generated unique id. This id can be used for serializing.

  2. usePercentages : boolean;

    3. If we set "usePercentages" to true the component will treat x and y values as percentages and will convert them into pixels before use. However if "usePercentages" is set to false then the component x and y will be treated as pixels.

  3. drawLayer : DrawLayer;

    4. Component are drawn into three layers:
    1. Background Layer
    2. Middleground Layer
    3. Foreground Layer
    While drawing the engine will first draw all the Background Layer components and then Middleground layer components and finally the Foreground layer components.

  4. duration :number;

    5. This value will decide the length / duration of display of a component. The end time of a component is calculated as start time + duration.
    When we append or add a component to the engine we have to provide its duration as well. duration may be converted to a method

  5. alwaysOn: boolean;

    6. docs being written

  6. version :string;

    7. docs being written

Public Methods

    Basic Methods

  1. update(msDelta :number,p :Pack):boolean; //second not frame

  2. draw(p :Pack):boolean;

    3. Time Methods

  3. getStartTime(inMilliSec ?:boolean) :number;

  4. setStartTime(n :number) :number;

  5. getEndTime(inMilliSec ?:boolean) :number;

    6. Animation X and Y

  6. goto(atFrame :number,x :number , y :number):boolean;

  7. animate(timeFrom :number,timeTo :number, xFrom :number,xTo :number, yFrom :number,yTo :number, ):boolean;
