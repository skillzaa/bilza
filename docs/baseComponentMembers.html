Base Component

BaseComponent is the Parent class for all components. It has following methods and public members:

    We have 2 values xAlign , yAlign. These 2 classes are used to align the X and Y coordingates of a component

  1. public xAlign :XAlignOpt;
  2. public yAlign : YAlignOpt;

    3. We have 2 values xRotate and yRotate. These 2 classes are used to align the rotate X and rotate Y coordingates of a component

  3. public xRotate :XAlignOpt;
  4. public yRotate :YAlignOpt;

    5. Both of the above 2 groups of alignment classes uses the following 2 options.

  5. public readonly XAlignOpt :typeof XAlignOpt;
  6. public readonly YAlignOpt :typeof YAlignOpt;

    7. Every component has a random id

  7. public readonly id :string;
  8. public usePercentages : boolean;

    9. The draw layer

  9. public drawLayer : DrawLayer;
  10. public style:Style;

    11. This will be set by the engine so do not touch it.

  11. public alwaysOn: boolean;
  12. public version :string;

    13. IMPLEMENTED IN ENGINE-- visible = false will not be drawn

  13. public visible :AniBoolean;

    14. implemented in Base Component but component creator has to implement it

  14. public rotation :AniNumber;

    15. implemented in Base Component but component creator has to implement it

  15. public opacity :AniNumber;

    16. The color prop points to the color of the main content

  16. public color :AniColor;

    17. This may or may not be implemented in some special cases like text which uses fontSize. However this will be used in 99% of the components

  17. public width :AniNumber;

    18. no need to convert it into AniNoPers

    same as width

  18. public height :AniNumber;

    19. no need to convert it into AniNoPers

    this will come in handy during using the lib with gui and while making interactive tools.

  19. public selected :boolean; //dont complicate.

    20. Padding and Border

  20. public border :AniNumber;
  21. public paddingTop :AniNoPerc;
  22. public paddingBottom :AniNoPerc;
  23. public paddingRight :AniNoPerc;
  24. public paddingLeft : AniNoPerc;

    25. Basic colors

  25. public colorBackground :AniColor;
  26. public colorBorder :AniColor;
  27. public showBackground :AniBoolean;
