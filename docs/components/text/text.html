Text Demo

On of the most basic component of bilza library is the Text component.ALmost all of the text displayed on the screen is done using the Text component.

A text component mainly comprises of margin, border, padding and the text content

Text Component Data

Every component has its data avaialble in a variable called data (or d). This variable is accessed using compName.d.dataValue.

  1. padding :number;

    2. padding: is the internal gap between the text and the border.

  2. margin :number;

    3. margin: is the gap between the X value of the component and the border of the component. As padding is internal gap, margin is the external gap that a component maintains.

  3. border :number;

    4. border: The width of border.

  4. fontSize :number;

    5. fontSize: is the fontSize of the text, increasing fontSize will increase the size of the text and vice versa. However fontSize is only used if flagUseDynResize is false. if flagUseDynResize is true then when ever the component is init (initialized) it will reset its fontSize based on dynWidth and dynHeight.

  5. dynWidth :number;(0 to 100)

    6. dynWidth: This is only use when flagUseDynResize = true. If flagUseDynResize = true then on init the component will dynamically change the fontSize to fit the width of canvas according to dynWidth. dynWidth represents the width of the canvas which the text component should fit in.

  6. dynHeight :number;(0 to 100)

    7. dynHeight is only used if flagShrinkHeightToFit = true.

    dynHeight: represents the canvas height which should fit in the text component if flagShrinkHeightToFit = true.

    When flagShrinkHeightToFit = true then on init The component will first fit the text according to dynWidth, once that is done the component will shrink the text such that it fits into the canvas height as mentioned by dynHeight.

  7. content :string;

    8. The content of the component.

  8. color :string;

    9. The color of the text

  9. colorBorder :string;

    10. The color of the border;

  10. colorBg :string;

    11. The background color.

  11. flagDrawContent :boolean;

    12. we can show or hide the text content using this flag

  12. flagDrawBorder :boolean;

    13. We can hide or show border using flagDrawBorder. Even when a border is not drawn its width is taken into consideration. If you want to totally remove the width of border from the calculation you have to set txt.d.border = 0.

  13. flagDrawBg :boolean;

    14. We can show or hide background using flagDrawBg

  14. flagTextShadow :boolean;

    15. We can hide or show text shadow using flagTextShadow

  15. flagBgShadow :boolean;

    16. We can hide or show background shadow using flagBgShadow

  16. flagUseDynResize:boolean;

    17. If flagUseDynResize = true the component will not use its fontSize rather on init will reset the fontSize to fit the text into portionof canvas = dyWidth

  17. flagShrinkHeightToFit:boolean;

    18. If flagShrinkHeightToFit = true then the component on init will check if the text component is taking more height of canvas than mentioned by dynHeight. If true it will be shrunk to fit the given height as per dynHeight.

  18. flagUseRelativeXY:boolean;

    19. We can set the x and y value

  19. maxDisplayChars :number; //implemented

    20. We can restrict the number of characters shown by the text component

  20. xAlignment :XAlignment;

    21. xAlignment will decide which point of the component is aligned to the x value of the component. The optins available are Left, Mid, Right.

  21. yAlignment :YAlignment;

    22. yAlignment will decide which point of the component is aligned to the y value of the component. The optins available are Top, Mid, Bot.
