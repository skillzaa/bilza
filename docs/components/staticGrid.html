StaticGrid

The Grid component uses fixed measurements (if you want to create a grid based on percentages use Grid).

  1. cellWidth :number;

    2. The width of each grid cell.

  2. cellHeight :number;

    3. The height of each grid cell.

  3. colorHorizontalLines :string ;

    4. The color of Horizontal Lines

  4. colorVerticalLines :string ;

    5. The color of Vertical Lines

  5. colorNumbers :string ;

    6. The color of the grid numbers.

  6. flagDrawNumbers :boolean ;

    7. We can make the grid numebers appear or disappear.

  7. flagDrawHorizontal :boolean;
  8. flagDrawVertical :boolean;
  9. lineWidthVertical :number;

    10. The width of the vertical line.

  10. lineWidthHorizontal :number;

    11. The width of the Horizontal line.

  11. fontSize :number;
  12. lineDash :number[];

    13. It takes an array of 2 number indicating width of line and the width of dash.
